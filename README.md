# Java JSP Book Rental System
<img src="./images/banner.png"/>

Management of book rental.

## Used Tools
- Java Server Pages (JSP)
### Web Server
- Apache Tomcat
### Database
- MS SQL Server 

## Features
- user self registration
- CRUD for books, users, categorys.
- Delayed rental time checking.
